# Feature: Setting an Alarm on a Smartwatch

## Step-by-Step Procedure

### Step 1: Power On the Smartwatch
Press and hold the power button until the smartwatch screen turns on.

### Step 2: Access the Applications Menu
Swipe on the home screen to open the applications menu available on the smartwatch.

### Step 3: Open the Alarm Application
Locate and select the Alarm application from the list of apps.

### Step 4: Create a New Alarm
Tap on the Add Alarm or + icon to create a new alarm.

### Step 5: Set the Alarm Time
Choose the required hour and minute using the on-screen controls.

### Step 6: Save the Alarm
After selecting the time, tap the Save button to confirm the alarm settings.

### Step 7: Verify Alarm Activation
Ensure that the alarm toggle is enabled so the alarm can function properly.
